How it works (for engineers)

Retrieval-augmented generation (RAG): each question retrieves the most relevant passages from a knowledge base via semantic embeddings; the model answers only from those passages and cites them, with guardrails and an independent red-team. Stack: vanilla-JS front end · serverless API on Vercel · Google Gemini (embeddings + chat) · honest “not in the knowledge base” fallback.
