# Integrations

_Category: Product · Last updated: 2026-04-15_

## Available integrations

Nimbus integrates with Slack, Microsoft Teams, Google Workspace (Drive and Calendar), GitHub, GitLab, Jira (import), Zapier, Zoom, and Figma.

## Connecting an integration

To connect an integration, go to Settings, Integrations, choose the app, and authorize it. Most integrations require a workspace admin to approve the connection.

## Slack troubleshooting

If Slack notifications stop arriving, check Settings, Integrations, Slack. If it shows Reauthorize, the token expired, often after a Slack permission change; click Reauthorize and approve in Slack to restore it.

## API & webhooks

A REST API and webhooks are available on Pro and above for building custom integrations.
